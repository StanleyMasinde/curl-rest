# Security Policy

## Supported Versions

We support the latest released version of this crate. If you are using an older version, please upgrade before reporting.

## Reporting a Vulnerability

Please report security issues **privately**.

- Email: hello@stanleymasinde.com
- Please include a minimal proof of concept, affected version(s), and any relevant logs.

We aim to respond within 72 hours and will coordinate a fix and disclosure timeline.

## Disclosure

We follow coordinated disclosure practices. If you are unsure whether an issue is security-related, please still reach out privately.
